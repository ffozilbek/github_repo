# GitHub Random Repository

A simple React app that fetches random GitHub repositories based on the selected programming language. Users can explore repositories, view details such as stars, forks, descriptions, and discover projects from different technologies.

## Features

- Select a programming language
- Fetch random GitHub repositories
- View repository details
- Display stars and forks count
- Loading and error states
- Powered by GitHub API and React Query

## Project Idea

https://roadmap.sh/projects/github-random-repo

## Tech Stack

- React
- TanStack Query
- Axios
- Tailwind CSS
- GitHub REST API
